# Rest Client in VS Code 
#TIL/dev/web/

1. Postman 처럼 API를 테스트해볼 수 있다. VS Code에서 매우 편리하게 테스트할 수 있어서 꽤나 유용할 듯. 
2. 사용법도 굉장히 쉽다. `.http` 혹은 `.rest` 파일을 만들고, GET/ POST url 을 적으면 **Send Request** 버튼이 생기는데 그거 클릭하면 끝.
