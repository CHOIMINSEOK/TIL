# Tuple
#TIL/dev/language/typescript

매우 간단하다. number와 string을 매핑하고 싶다면, 그냥 이렇게하면된다. 

```typescript
var example: [number, string]

//init
example = [1, "호이"];

log example[0] // 1
log example[1] // "호이"

```
